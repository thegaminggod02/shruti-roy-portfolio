# Shruti Roy - Professional Portfolio Website

An interactive, responsive, and modern developer portfolio website designed for **Shruti Roy**, specializing in **AI & Data Science** and **Android Development**.

## 🚀 Features

- **Dynamic Hero Canvas**: Interactive mouse-repelling particle network constructed on HTML5 Canvas.
- **Glassmorphism Theme (AI Nexus)**: Futuristic dark UI aesthetic with high-performance CSS blur filters.
- **Dark/Light Theme Toggle**: Seamless transition system saving preferences to local storage.
- **Micro-interactions**: Subtle CSS hover-lifts, Custom circular orbit animation, cursor glows, page scroll progress, and button ripple click responses.
- **Timeline Milestones**: Compact educational and professional experience vertical roadmaps.
- **Responsive Adaptations**: Tailored layouts across custom desktop, tablet, and mobile breakpoints.
- **Interactive Form Validation**: Client-side validation with async submission simulation feedback.
- **Clean SEO / Open Graph Setup**: Complete index meta attributes for improved indexing and sharing.
